# 白噪音音频文件目录

## 使用说明

将高质量的 `.mp3` 或 `.wav` 白噪音音频文件放在此目录下，应用会自动检测并使用真实音频（优先于程序生成的白噪音）。

## 文件名（必须与以下名称完全一致）

| 文件名 | 对应白噪音 | 推荐时长 |
|--------|-----------|---------|
| `ocean.mp3` | 近海潮声 | 3-10 分钟 |
| `stream.mp3` | 山间溪流 | 3-10 分钟 |
| `rain.mp3` | 细雨滴答 | 3-10 分钟 |
| `wind.mp3` | 林间晚风 | 3-10 分钟 |
| `campfire.mp3` | 篝火 | 3-10 分钟 |
| `tea.mp3` | 煮茶 | 3-10 分钟 |

## 免费音频资源推荐

以下网站提供免费、高质量的 CC0 白噪音音频：

1. **[Pixabay Sound Effects](https://pixabay.com/sound-effects/)** — 搜索 "ocean waves", "rain", "campfire" 等
2. **[Freesound.org](https://freesound.org/)** — 筛选 CC0 许可
3. **[Mixkit Sound Effects](https://mixkit.co/free-sound-effects/)** — 免费高质量音效
4. **[YouTube Audio Library](https://www.youtube.com/audiolibrary)** — 免费音效

## 注意

- 音频文件会自动循环播放
- 如果没有放置音频文件，应用会使用内置的程序生成白噪音
- 建议使用 128-320kbps 的 MP3 格式以平衡音质和加载速度
